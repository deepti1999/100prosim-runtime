Was bedeutet das?

Jeder Admin-Nutzer kann einer Rolle zugeordnet werden. Die Rolle entscheidet, ob diese Person nur schauen, bearbeiten, Szenarien wiederherstellen oder alles verwalten darf.

Wichtig: Ein Nutzer muss zusätzlich Mitarbeiter-Status haben, damit er die Django-Verwaltung öffnen kann.

So gibst du jemandem eine Rolle

  1. Auf Rolle zuweisen klicken.
  2. Person auswählen.
  3. Passende Rolle auswählen.
  4. Speichern.

Die Seite setzt den Mitarbeiter-Status automatisch, damit die Person die Django-Verwaltung öffnen kann.
