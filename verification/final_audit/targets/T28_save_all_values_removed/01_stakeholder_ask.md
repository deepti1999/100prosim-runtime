# T28 — Save All Values button removed

**PDF:** §2.4.5 (page 5), lines 202–211. *"Beim aktuellen 100prosim-Web wird den Anwendenden auf der Seite 'Flächen' der Button 'Save All Values' angeboten. … Wenn dem so ist, ist der Button 'Save All Values' überflüssig und unnötig verwirrend."*

**English:** "Save All Values" button on the Flächen page is redundant with `Scenarios → Save current Scenario`, only covers LandUse (not Renewables/Verbrauch), confusing. Remove.

**Acceptance:** Button absent from `/landuse/`. Underlying scenario-save mechanism preserved.
