# T8 — Parameter source (Quellbezug) surfaced in UI

**PDF §2.3.1** (page 3, lines 89–99): *"In 100prosim-Excel sind für jeden Parameter-Ansatz die Quellbezüge und getroffenen Annahmen einfach und direkt über Verlinkung nachvollziehbar."*

**Acceptance:** every parameter row in the UI exposes its source_url for click-through.
