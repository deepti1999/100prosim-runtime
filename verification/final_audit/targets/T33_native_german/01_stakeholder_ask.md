# T33 — Native German (not Google-translate)

**PDF §2.5.1** (page 6): *"Bei der Nutzung einer Google-Übersetzung … kommt es zu fehlerhafter Übersetzung von einzelnen Begriffen und Button-Bezeichnungen."*

**English:** Google-translate produces wrong terms. Use native German throughout.

**Acceptance:** No Google-translate-style awkward phrasings; use German energy-domain conventions; no untranslated English UI strings remaining.
