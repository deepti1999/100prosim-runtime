# T34 — German number format (display)

**PDF §2.5.2** (page 6, lines 231–237): *"Im aktuellen 100prosim-Web werden die Zahlenwerte im englischen Zahlenformat dargestellt (Komma für Tausender-Gruppierung, Punkt als Dezimaltrennzeichen). Für Anwendende aus dem deutschen Sprachraum kann das sehr verwirrend sein, da sie genau die umgekehrte Trennzeichenverwendung gewohnt sind (Punkt für Tausender-Gruppierung, Komma als Dezimaltrennzeichen)."*

**English:** Numbers in English format. Switch to German: dot for thousands, comma for decimals.

**Acceptance:** All numeric displays use `.` thousands + `,` decimal across every page.
